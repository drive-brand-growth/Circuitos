1. **Comprehensive Knowledge Bases Drive Agent Value**
   - V2 covered only 19% of revenue
   - V3 covers 100% of revenue
   - Lesson: Complete data = complete revenue coverage

2. **ML Integration Starts with Data Documentation**
   - V3 documents all ML model inputs
   - Enables rapid model development
   - Lesson: Document ML requirements early

3. **High-Intent Detection Multiplies Lead Value**
   - Licensing leads worth $40k-60k each
   - Founder's memberships worth $2,500 each
   - Lesson: Not all leads are equal - prioritize high-value intents

4. **Single Agent Can Handle Multiple Revenue Streams**
   - Events AI agent now handles 7 revenue streams
   - Specialized agents will optimize each stream
   - Lesson: Start broad, then specialize
