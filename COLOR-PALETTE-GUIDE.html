Quick Reference Guide

Use Neon Green For:

  • Active/selected states
  • System operational status
  • Real-time indicators
  • Confidence scores > 85%
  • Progress bars
  • Primary CTAs
  • Interactive hover states
  • Success confirmations

Use Gold For:

  • Logo frames & branding
  • Revenue/money statistics
  • Premium tier badges
  • Executive titles
  • Certification badges
  • High-value indicators
  • Luxury features
  • Brand typography (titles)

Use Dark Knight For:

  • All page backgrounds (100%)
  • Card containers
  • Navigation panels
  • Modal backgrounds
  • Content sections
  • Footer areas
  • Sidebar panels
  • Base UI layer
